Uniform Resource Locator (URL)

The "You Are Here" map marker.

A URL is a specific address for a specific file. It tells the browser exactly which server to go to and which folder to look in. It's the difference between saying "Go to the library" and saying "Go to the library on 5th Street, 3rd floor, 2nd shelf, 4th book from the left."
