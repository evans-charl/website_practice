Domain Name Server (DNS)

The internet's GPS.

Computers hate names; they only like numbers (IP addresses). Humans hate numbers; we only like names. The DNS is the middleman. When you type "google.com," the DNS whispers the number "142.250.190.46" into the computer's ear so it knows where to go. It's the ultimate digital translator.
